# الصافي — Safi

## مقدمة

«الصافي» تطبيق جوال لإدارة الماليات اليومية: المحافظ، والدخل والمصروف، والمعاملات، ودفتر الديون، والعملاء، مع دعم اللغة العربية واتجاه العرض من اليمين إلى اليسار. هذا الملف موجّه لأي قارئ يريد فهم المشروع أو تشغيله أو المساهمة فيه دون افتراض خبرة عميقة بالبرمجة.

## ما الذي يقدّمه التطبيق؟

- توحيد عرض الأرصدة عبر أكثر من محفظة (نقد، بنك، محافظ إلكترونية، وغيرها).
- تسجيل حركات الدخل والمصروف مع الملاحظات والتصنيف والمرفقات عند الحاجة.
- متابعة دفتر الديون والعملاء والمستحقات وسجل السداد.
- أرشفة المعاملات والاطلاع على التقارير والمخرجات مثل ملفات PDF عند استخدام الميزات المعتمدة في التطبيق.
- تنبيهات محلية (إشعارات) لمواعيد أو تذكيرات مرتبطة بالإعدادات.
- تسجيل دخول عبر الهاتف مع حماية البيانات في بيئة Firebase.

## البيئة التقنية (باختصار)

- **لغة التطبيق:** Dart
- **إطار العمل:** Flutter (إصدار SDK المذكور في المشروع: 3.11.4 فما فوق)
- **الواجهة:** Material Design، ومكتبات مساعدة للرسوم (مثل المخططات البيانية)
- **إدارة الحالة:** Riverpod
- **الخلفية السحابية:** Firebase (النواة، المصادقة، قاعدة بيانات Firestore)
- **تخزين محلي ووظائف أخرى:** تفضيلات المستخدم، جهات الاتصال (بإذن المستخدم)، اختيار الصور، التحقق من الاتصال، الطباعة والمشاركة، إشعارات محلية مع المنطقة الزمنية

التفاصيل الكاملة للحزم البرمجية مذكورة في ملف `pubspec.yaml`.

## متطلبات التشغيل على الجهاز

1. تثبيت Flutter SDK ومحرر نصوص أو بيئة تطوير (مثل Android Studio أو VS Code مع إضافة Flutter).
2. إعداد مشروع Firebase وربطه بالتطبيق: وجود ملف `google-services.json` ضمن مشروع أندرويد، وملف `GoogleService-Info.plist` لمشروع iOS عند البناء لهذه المنصة.

## تشغيل المشروع محلياً

بعد فتح المجلد الرئيسي للمشروع في الطرفية:

```bash
flutter pub get
flutter run
```

لبناء نسخة إصدار (Release) لأندرويد على سبيل المثال:

```bash
flutter build apk --release
```

ملف الـ APK الناتج يظهر عادة ضمن `build/app/outputs/flutter-apk/` وفق إعدادات Gradle في المشروع.

## ملاحظة للمختبرين والمطورين

في بيئة التطوير أو الاختبار قد يُستخدم **رمز تحقق افتراضي هو `123456`** عند إكمال خطوة التحقق من الهاتف. يُنصح بتغيير هذا السلوك في الإنتاج الفعلي وفق سياسة الأمان المعتمدة لديكم.
